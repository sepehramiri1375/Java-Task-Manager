# Java-Task-Manager
Simple Java Task Manager
## Features
- Add tasks
- Show all tasks
- Mark tasks as done
- Delete tasks

## Technologies
- Java
- Object-Oriented Programming
- ArrayList
- Scanner

## Project Structure
- Main.java
- Task.java
- TaskManager.java

## How to Run
1. Open the project in IntelliJ IDEA
2. Run Main.java
3. Use the console menu to manage tasks

## Example Menu
1 - Add task
2 - Show tasks
3 - Mark task as done
4 - Delete task
5 - Exit
